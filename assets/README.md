# DJOULE DROPSHIPPING - Instructions pour les Images

## Structure des Assets

Le dossier `assets/` contient :

### `/assets/images/`
Images pour le site web :
- Photos du fondateur Ibrahim Saleh
- Photos du marché de Yiwu
- Photos des produits par catégorie
- Photos de l'équipe sur le terrain
- Images d'illustration pour les pages

### `/assets/videos/`
Vidéos pour le site web :
- Vidéo de présentation du fondateur
- Vidéo de visite du marché Yiwu
- Vidéos de démonstration des produits
- Témoignages clients

### `/assets/documents/`
Documents pour le téléchargement :
- Catalogue produits (PDF)
- Certifications et documents officiels
- Présentation de l'entreprise (PPT)
- Fiches techniques produits

## Images Recommandées

### Photos Fondérateur
- Portrait professionnel d'Ibrahim Saleh
- Photos sur le terrain au marché
- Photos avec des fournisseurs
- Photos avec des produits

### Photos Marché Yiwu
- Vue d'ensemble du marché
- Allées du marché
- Intérieurs d'entrepôts
- Atmosphère commerciale

### Photos Produits
- Matériel de construction
- Quincaillerie et outils
- Matériel électrique
- Plomberie et sanitaires
- Équipements de sécurité

### Spécifications Techniques
- **Format**: WebP ou JPG
- **Taille**: Max 2MB par image
- **Dimensions**: 1920x1080px (hero), 800x600px (standard)
- **Qualité**: 80-90% compression
- **Alt text**: Description détaillée pour SEO

## Vidéos Recommandées

### Contenu Vidéos
- Présentation du fondateur (1-2 min)
- Tour du marché Yiwu (3-5 min)
- Démonstration produits (30-60s par produit)
- Témoignages clients (1-3 min)

### Spécifications Techniques
- **Format**: MP4 (H.264)
- **Résolution**: 1080p minimum
- **Bitrate**: 5-8 Mbps
- **Audio**: AAC 128kbps
- **Durée**: Optimisée pour web

## Placeholder Images

En attendant les vraies photos, vous pouvez utiliser :
- Photos de stock gratuites (Unsplash, Pexels)
- Placeholders avec couleurs de la marque
- Icônes et illustrations vectorielles

## Optimisation

1. **Compression**: Utiliser TinyPNG ou ImageOptim
2. **Lazy Loading**: Implémenté dans le JavaScript
3. **Formats Modernes**: WebP avec fallback JPG
4. **CDN**: Configurer pour production
